# 🤖 Простой Chatbot - Пример для начинающих

Это простой пример chatbot-а написанный на Python. Идеально для первого проекта!

## 📋 Описание

Простой чат-бот, который отвечает на вопросы по предопределённым шаблонам. Он может:
- ✅ Отвечать на приветствия
- ✅ Рассказывать о себе
- ✅ Давать простые ответы
- ✅ Сказать пока

## 🚀 Быстрый старт

```bash
# 1. Установка зависимостей (если они есть)
pip install -r requirements.txt

# 2. Запуск
python main.py

# 3. Пиши вопросы!
```

## 📊 Примеры использования

```
Введи что-то: привет
Chatbot: Привет! Как дела? 👋

Введи что-то: что ты умеешь?
Chatbot: Я простой chatbot, могу отвечать на вопросы!

Введи что-то: пока
Chatbot: До встречи! 👋
```
