# 🤖 Мои AI Проекты

Репозиторий для хранения моих маленьких AI и машинного обучения проектов.

## 📂 Структура репозитория

```
.
├── README.md                # Этот файл
├── .gitignore              # Что НЕ загружать в GitHub
├── docs/                   # Документация и гайды
│   └── getting-started.md  # Инструкция для новичков
├── projects/               # Основная папка с проектами
│   └── simple-chatbot/     # Пример первого проекта
└── template/               # Шаблон для новых проектов
    ├── README.md
    ├── main.py
    └── requirements.txt
```

## 🚀 Быстрый старт

### Для запуска существующего проекта:

```bash
# 1. Зайди в папку проекта
cd projects/your-project-name

# 2. Установи зависимости
pip install -r requirements.txt

# 3. Запусти проект
python main.py
```

### Для добавления нового проекта:

1. Создай папку в `projects/`
2. Скопируй файлы из `template/` 
3. Отредактируй под свой проект
4. Загрузи в GitHub (см. инструкцию ниже)
